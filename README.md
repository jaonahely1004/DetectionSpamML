## Stack Technologique
- Langage : Python 3.18
- Librairies ML : Scikit-Learn, Joblib, Pandas, numpy, scipy,nltk, flask
- Interface : Streamlit
- Déploiement : Streamlit Cloud

## Processus et Modèles
- Dataset : Huggingface Multilingue
- Traitement : Tokenisation,suppression de Stop Word, Mise en minuscule, suppression ponctuation et espace vide.
- Vectorisation : TF-IDF (Term Frequency-Inverse Document Frequency).
- Modèle ML : RandomForestClassifier.
